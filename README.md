# LimenServe Frontend

This repository now contains the frontend application only.

## App

- `web-app`: React + Vite frontend

## Frontend stack

- React
- Vite
- Three.js
- CSS
- PWA support

## Development

From `web-app`:

```powershell
npm install
npm run dev
```

## Build

From `web-app`:

```powershell
npm run build
```

## Deployment

Deploy the `web-app` folder to Vercel.

Required frontend environment variables:

- `VITE_API_URL`
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`
